# go_unit_test

A Go unit test project for the school. 

# How to install

- Install GO : https://golang.org/doc/install
- Install testify : https://github.com/stretchr/testify
- Install gomock : https://github.com/golang/mock (mockgen is not necessary) 
- Clone this repo
